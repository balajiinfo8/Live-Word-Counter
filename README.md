Project Title
Live Character Count (Excluding Spaces)

Project Description
This is a beginner-friendly web application that lets users count the number of characters they have typed in an input field, while ignoring all white spaces. As you type or edit your text, the character count updates instantly, providing real-time feedback. This project is useful for text analysis, copywriting, or just practicing JavaScript and DOM manipulation.

Features
Live Counting: The character count updates live as the user types.

Excludes Spaces: All whitespace characters (spaces, tabs, etc.) are ignored in the count.

Simple Interface: Minimalist and responsive design for easy use.

Accessible Demo: Deployed with GitHub Pages so anyone can try it online.
